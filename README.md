Project 1 
CYP450 Pharmacogenomics Analysis

Overview
This project simulates pharmacogenomic effects on drug metabolism using DNA-to-protein translation and mutation analysis.
Methods
- GC content calculation of gene sequences
- DNA to protein translation using Biopython
- Single nucleotide mutation simulation
- Predicted effects on enzyme structure and function

 Significance
Demonstrates how genetic variations (SNPs) can alter enzyme activity and influence drug metabolism, connecting pharmacy knowledge with computational biology.
 Tools
- Python
- Biopython
 Highlights
- Simulated SNPs and their impact on protein sequences
- Modeled metabolizer phenotype classification
- Reinforces understanding of pharmacogenomics in drug response
