# CYP450 Pharmacogenomics Analysis

## Overview
This project simulates pharmacogenomic effects on drug metabolism using DNA-to-protein translation and mutation analysis.

## Methods
- GC content calculation
- DNA to protein translation
- Mutation simulation
- Functional interpretation

## Tools
- Python
- Biopython

## Significance
This project demonstrates how genetic variations influence drug metabolism and connects pharmacy knowledge with computational biology.
